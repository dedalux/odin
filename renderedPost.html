Setting up my personal blog

05 April 2016 : blog

After starting to get my hands on learning javascript development, I decided to start this blog to track my learning progress. On a whim I registered a trendy .io domain and set up Ghost with DigitalOcean. The setup was fairly painless and the blog engine was up and running in a minute.

However, I encountered some slight set-back when I tried to connect my custom domain. I edited the config.js in Ghost with my domain url as directed, yet as I refreshed the page countless times the change still didn't take place. All my page links were still the default Ghost domain.

Turns out, the issue was that I needed to restart Ghost on the server. Lesson learned for next time.

I still have some customization to do, perhaps add a few such as extensions such as analytics. More to come.
